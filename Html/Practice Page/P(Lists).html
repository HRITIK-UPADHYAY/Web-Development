Block Element

Block Element : Those Element Which is start from new Line and it will takes all the Width of the Device.

    Examples

  1. P Tag
  2. Headings Tag
  3. Div Tag
  4. Section Tag

Inline Element

Inline Element : Those Element which is start from point where previous is ended.

    Examples

  • Span Tag
  • Audio Tag
  • Video Tag
  • Image Tag
