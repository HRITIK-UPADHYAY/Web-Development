CSS Transition: allow you to change property value smoothly over a given duration.

  1. transition
  2. transition-delay: it specifies when will the transition effect start.
  3. transition-duration: it specifies that for how much time the effect should last.
  4. transition-property: it specifies the CSS property to which you want to apply transition property to.
  5. transition-timing-function: it specifies the speed curve of the transition effect.
    1. ease, linear, ease-in, ease-out, ease-in-out, cubic-bezier(n,n,n,n)
