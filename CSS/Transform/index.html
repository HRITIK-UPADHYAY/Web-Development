1. translate(x-axis, y-axis): it moves an element from its current position w.r.t X and Y axis value given.
  2. rotate(deg): it rotates your element counter clockwise.
  3. scale(x-axis, y-axis): it increases and decreases the size of the element.
  4. skew(deg): it skews an element along the X and Y Axis.
  5. matrix(scaleX, skewY, skewX, scaleY, translateX, translateY): combines all the transform method into one.
